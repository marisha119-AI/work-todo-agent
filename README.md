# ✅ Work To‑Do Agent

An AI‑powered task manager that lets you add, list, complete, and delete tasks using natural language. Built with Streamlit and the OpenRouter API (model: `z-ai/glm-4.5-air:free`).

![Demo](demo/screenshot.png)

## Features
- ✍️ Add tasks: "Add task Prepare presentation by Friday high priority"
- 📋 List tasks: "Show my tasks"
- ✅ Mark as done: "Complete task a1b2c3"
- 🗑️ Delete tasks: "Delete task xyz789"
- 🎨 Clean, modern UI with priority colour coding
